# Task-3-SQL-for-Data-Analysis
This project performs SQL-based data analysis on a custom Ecommerce dataset. The dataset contains three tables: customers, products, and orders. The objective is to write SQL queries using SELECT, WHERE, GROUP BY, JOINs, subqueries, views, and indexing to derive insights from the data.

# Dataset used
# 1. customers.csv

Contains customer information
Columns:

customer_id

customer_name

email

city

# 2. products.csv

Contains product details
Columns:

product_id

product_name

category

price

# 3. orders.csv

Contains order transactions
Columns:

order_id

customer_id

product_id

order_amount

order_date

# SQL Tasks Performed

SELECT statements

WHERE filtering

ORDER BY sorting

GROUP BY with aggregates (SUM, AVG)

INNER JOIN, LEFT JOIN, RIGHT JOIN

Subqueries

Creating Views

Query Optimization using Indexes
